系统要求

确保您的设备满足运行条件，获得最佳使用体验

Windows 系统

  • Windows 10 (64位) 或更高版本
  • 至少 4GB 系统内存
  • 500MB 可用存储空间
  • 支持 DirectX 11 显卡

macOS 系统

  • macOS 10.15 Catalina 或更高版本
  • 至少 4GB 系统内存
  • 500MB 可用存储空间
  • 支持 Intel 或 Apple Silicon 处理器

Linux 系统

  • Ubuntu 18.04 / CentOS 8 或更高版本
  • 至少 4GB 系统内存
  • 500MB 可用存储空间
  • 支持 GLIBC 2.27+ 运行库

专业提示

新手推荐：

选择 u2net 或 silueta 模型，体积小速度快，适合日常使用

追求效果：

使用 rmbg-1.4 或 bria-rmbg-2.0 模型，效果更佳但需要更多内存

批量处理：

处理大批量图片时建议降低优化等级，避免内存不足

关于 Jevet

Jevet 是一款完全免费开源的 AI 图片处理工具，基于 Electron 和 Vue3 开发，致力于为用户提供专业、安全、高效的图片处理体验。
